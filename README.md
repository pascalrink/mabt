# mabt

Pascal Rink & Werner Brannath (2025). Post-selection confidence bounds for prediction performance. Machine Learning, Volume 114, 82, Springer Nature. URL: https://link.springer.com/article/10.1007/s10994-024-06632-w

Status: **Archived** (research artifact).

Further development continues in: https://github.com/pascalrink/prediction-performance-ci
